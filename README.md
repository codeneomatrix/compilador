# COMPILADOR PASCAL
Compilador del leguaje pascal echo en julia

### uso
En la ubicacion de los archivos se abre una consola y se escribe:
```julia
 julia ANALIZADOR.jl
```
aprecera en la consola lo siguiente:
 ```julia
 Url del archivo .pas:
 {Escribe la url del archivo con extension .pas}
 ......
```
